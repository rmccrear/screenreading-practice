Windows Narrator

See reference: [ LINK ]

To launch Narrator, press the Windows logo key + Ctrl + Enter. To stop the narrator, press these keys once more.

Another way to launch Narrator: Press the Windows logo key + Ctrl + N to open Narrator settings, and then turn on the toggle under Use Narrator.

After launching Narrator, you can enter or exit SCAN MODE with CAPS LOCK + SPACEBAR

In SCAN MODE, use the TAB button or the ARROW KEYS to move. (Use SHIFT + TAB to move backwards.)

In SCAN MODE, use the SPACEBAR to "click" a button.

Windows Narrator

See reference: [ LINK ]

To launch Narrator, press the Windows logo key + Ctrl + Enter. To stop the narrator, press these keys once more.

Another way to launch Narrator: Press the Windows logo key + Ctrl + N to open Narrator settings, and then turn on the toggle under Use Narrator.

After launching Narrator, use TAB or the arrow keys to move. (SHIFT + TAB to move back.)
